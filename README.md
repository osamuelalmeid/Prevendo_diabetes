# Prevendo_diabetes
Definindo o Business Problem, necessitamos prever se uma pessoa irá desenvolver diabetes, então vou utilizar Dados Históricos e Machine Learning para realizar essa verificação.

# Requisitos
<li>Python 3.5</li>
<li>Pandas</li>
<li>Numpy</li>
<li>scikit-learn</li>
<li>NaiveBayes</li>
<li>RandomForest</li>
<li>RegressionLogistic</li>
